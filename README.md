# Half Marathon Training Calendar

A vibrant, responsive training calendar for the **80/20 Half Marathon Training 2026** plan.

## Run it

### Option A: Open directly
- Double-click `index.html` to open it in your browser.

### Option B: Local server (recommended)
From this folder:

```bash
python3 -m http.server 5173
```

Then open:
- http://localhost:5173

## Use it
- Set **Week 1 start (Mon)** to the Monday you want.
- Switch between **4 weeks**, **Month**, **Week**, and **Day**.
- Click/tap a workout to load its details in the side panel.
- Long-press a workout to open a popup with details.
